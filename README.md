# Sitio portfolio (estilo menú)

Proyecto en React + Vite. La interfaz está en español; los textos, enlaces, trabajos y habilidades se centralizan en un solo archivo para que puedas sustituirlos por los tuyos.

## Personalizar contenido

Edita **`src/siteContent.js`**: ahí están música de fondo referenciada, menú principal, sección «Sobre mí», experiencias laborales, proyectos, habilidades y redes. Sustituye los marcadores tipo `[TU NOMBRE]` o las URLs de ejemplo por tus datos.

Para cambiar imágenes y vídeos, coloca archivos nuevos en `src/assets/` y actualiza las rutas `import` al inicio de `siteContent.js`.

## Ejecutar en local

```bash
npm install
npm run dev
```

Compilar para producción: `npm run build`; vista previa del build: `npm run preview`.
